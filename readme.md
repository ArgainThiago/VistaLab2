PROYECTO SALUDLAB
Plataforma Web para los centros de Salub Publicos
Desarrollada por: Belén Tea, Thiago Argain, Ramiro Fonseca.
El proyecto consiste en el diseño y desarrollo de un software en formato de página web, cuyo objetivo es optimizar la gestión de consultas médicas tanto para pacientes como para profesionales de la salud. 
La aplicacion permite: Agenda personal para el medico agregar y eliminar cita. El medico tendra acceso al historial clinico del paciente, administrar su disponibilidad y tendra un chatbot integrado.
El paciente puede sacar número, cancelar cita con un hora de anicipación, acceso a su ficha medica basica, histoarial de citas pasadas/futuras y un chatbot integrado
Para el administrador: Permite gestionar paciente y medico.

Objetivo del Software:
Su objetivo general es desarrollar un software como aplicación para poder optimizar la gestión de recursos de atención al paciente; también con esta aplicación se podrá hacer que el médico especialista tenga mucho más orden en sus turnos. 

Objetivos específicos:
Diseñar una solución tecnológica que permita mejorar los turnos ya sea del paciente o médico.
Implementar un prototipo funcional del programa utilizando herramientas de programación como html, CSS, JavaScript, PHP.
Promover el uso de herramientas digitales que es lo que necesita  nuestro Centro de Salud.
Para ejecutar este proyecto se necesita:
XAMPP (Apache, MySQL, phpMyAdmin)
GitHub (Para descargar el repositorio)
Navegador web actualizado (Chrome, Edge, Firefox, etc)
